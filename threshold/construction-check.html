    Threshold Check

    Construction Check

    Before moving to Legacy: have I built something that matters?

    Have I built something that exists outside my own mind?

    In short: Construction asks three things: an artifact that exists externally, value that outlasts my reputation, and a system that runs without my constant presence.

    The Three Requirements

    The Externality Test

    Does what I've built exist outside my own mind? Code that runs. Words that can be read. A system that functions. Not an idea. Not a plan. An artifact.

    The Anonymity Test

    Would this matter if my name were removed from it? If the value depends on people knowing I built it, I have built visibility, not value. True contribution stands alone.

    The Independence Test

    Does this operate without my constant presence? A system that requires me to function is a job. Contribution creates things that continue without me—systems, ideas, paths others can walk.

    Counterfeit Construction

    “I performed building (visibility without value).”

    The GitHub commit followed by the tweet announcing the commit. The LinkedIn post about the grind. Productivity theatre that produces visibility instead of value.

    True Construction

    I have an artifact that exists outside my mind. Its value does not depend on my name. It continues without my constant presence. I have shipped the 70% version.

    The Antifragility Test

    “If no one was watching, would I still build this? Would this matter in ten years?”

    Construction is not about being seen building. It is about building things that stand on their own, whether I am seen or not.

    The Decision

    Before I proceed to Legacy, I ask myself honestly:

    • Can I point to something I've built that exists outside my own mind?
    • Would this thing still have value if my name were removed?
    • Does it operate without my constant presence?
    • Have I shipped the 70% version, or am I still perfecting the plan?

    If I answered “no” to any of these: I remain in Construction. I cannot leave a legacy if I have built nothing. Plans are not artifacts.

    If I answered “yes” to all: I am ready to consider what remains. Proceed to Legacy.
